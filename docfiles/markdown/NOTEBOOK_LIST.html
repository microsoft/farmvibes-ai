  • ADMAg Integration ğŸ““ : Connecting FarmVibes.AI and Microsoft Azure Data Manager for Agriculture (Data Manager for Ag)

  • Carbon sequestration evaluation with Azure Data Manager for Ag and COMET-Farm API ğŸ““ : Derive carbon sequestration information with COMET-Farm API for agricultural fields, using Azure Data Manager for Agriculture to retrieve farming data.

  • Crop cycle detection ğŸ““ : Run time-series analysis over NDVI data to detect the number of crop cycles in a year for a certain region.

  • Crop land segmentation (1/4) - dataset generation ğŸ““ : Generate a dataset based on NDVI and CDL rasters to train a crop land segmentation model.

  • Crop land segmentation (2/4) - dataset visualization ğŸ““ : Visual exploration of the crop land segmentation dataset.

  • Crop land segmentation (3/4) - local model training ğŸ““ : Train a crop land segmentation model locally.

  • Crop land segmentation (3/4) - model training on Azure Machine Learning ğŸ““ : Train a crop land segmentation model on Azure Machine Learning.

  • Crop land segmentation (4/4) - inference ğŸ““ : Infer crop land segmentation for new regions with a trained model.

  • Green House Gas fluxes ğŸ““ (30s) : Compute GHG emissions and sequestration for a given crop, location and time range.

  • Harvest and germination periods ğŸ““ : Infer harvest and germination periods from NDVI time series.

  • Hello World ğŸ““ : Simple example of how to instantiate a client and run a workflow.

  • Investigating Amazon Rainforest deforestation with SpaceEye ğŸ““ : Download and preprocess Sentinel data to obtain cloud-free images using SpaceEye over the Amazon Rainforest.

  • Land degradation ğŸ““ : Compute NDVI trends on Landsat images and analyse precipitation data to detect land degradation.

  • Micro climate prediction ğŸ““ : Train a model to predict micro climate parameters given global weather information and local weather station data.

  • Nutrient Heatmap Estimation ğŸ““ : Using soil samples from a farm, train a Random Forest model to estimate a heatmap of soil nutrients.

  • Spectral indices ğŸ““ : Compute and visualize spectral indices from Sentinel-2 imagery.

  • Timelapse visualization ğŸ““ : Generate a timelapse of SpaceEye and NDVI rasters.

  • Weed detection ğŸ““ : Train a Gaussian Mixture Model to group similar regions in a raster image affected by weed.

  • What-if analysis of carbon sequestration ğŸ““ : Derive carbon sequestration information with COMET-Farm API for agricultural fields.
